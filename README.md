# instagram_terraform
instagram style web architecture using terraform

JSCode의 AWS 입문 Finalproject 테라폼으로 구현

### 아키텍쳐 구성 
- Infra : Terraform, AWS(VPC,EC2,ALB,S3,SG), DB(RDS MySQL)
- Backend : Springboot REST API, S3(image)
- Frontend : Express, S3(정적 웹)
